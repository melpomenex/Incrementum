## Why

The user handbook currently states there are "17 built-in themes" — but the app actually ships **147 built-in themes** (26 modern/curated + 121 legacy). Additionally, the handbook doesn't document the font family selection feature at all, despite the app offering **65 fonts** across 5 categories. Users are getting inaccurate information about what's available to them.

## What Changes

- Update the Initial Setup section theme count (currently "17 built-in themes") to reflect the actual total (147)
- Update the Settings > Appearance > Themes section with accurate theme counts, broken down by modern vs legacy
- Add documentation for the font family selection feature, including the count (65 fonts across 5 categories: sans-serif, serif, monospace, display, system) and where to find it in settings
