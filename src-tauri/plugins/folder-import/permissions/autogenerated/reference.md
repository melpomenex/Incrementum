## Default Permission

Allows the folder-import plugin's folder + file picker commands to be invoked
from the frontend.

#### This default permission set includes the following:

- `allow-pick-folder-documents`
- `allow-pick-files`
- `allow-install-apk`
- `allow-backup-db-to-downloads`
- `allow-register-share-listener`
