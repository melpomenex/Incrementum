## Default Permission

Allows the folder-import plugin's folder picker command to be invoked
from the frontend.

#### This default permission set includes the following:

- `allow-pick-folder-documents`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`incrementum-folder-import:allow-pick-folder-documents`

</td>
<td>

Enables the pick_folder_documents command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`incrementum-folder-import:deny-pick-folder-documents`

</td>
<td>

Denies the pick_folder_documents command without any pre-configured scope.

</td>
</tr>
</table>
